Perl, XML, and WAP

Andy Murren

OVEN Digital

Thursday, 3:30 in Rangos III.

The basic outline for this talk is how we at Oven are using Perl with the XML, CGI and Apache modules and Apache's mod_perl to serve the same content for both http clients and WAP clients.

By June Oven will have completed a prototype and demo application for a client and will have its own Internet site fully WAP enabled.

Outline:

  1. An overview of how WAP works and how it relates to http
  2. Limitation of WAP enabled devices
  3. Using XML and XSL
  4. How CGI.pm XML:: and mod_perl are used
  5. Changes in web site development to support WAP
  6. Multimedia integration with wireless
  7. Development trends in wireless devices
