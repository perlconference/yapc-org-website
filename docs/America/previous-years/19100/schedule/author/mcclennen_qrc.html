QRC: perl-based software for coordinating an online reference service

Michael McClennen

Internet Public :Library

Friday, 2:15 PM in Rangos I.

QRC is a software package whose job is to coordinate the operation of an online reference service or help desk.  The project was inspired by Hypermail and Req, but incorporates many new ideas as well.  This software has now been in active use (and development) for about five years, and is deployed by several libraries throughout the world.  The QRC software serves as a good example of the kinds of projects that Perl is good for, and the code uses many advanced Perl techniques which I will point out and demonstrate.
