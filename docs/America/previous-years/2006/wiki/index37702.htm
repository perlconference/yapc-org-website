Connecting to IRC while at the conference

The problem with irc at IIT is that they block outbound port 6667 due to it's misuse by botnets, DDoS, etc. You'll have to use non-standard ports, ssh to an external server and irc fom there, or set up an ssh tunnel and run your irc client through that. If you still can't do that, JoeyKelly has set up a redirect on his machine back home to get around the firewall. Just point your irc client at home.joeykelly.net, on port 443 (SSL). It works... you'll automagically end up on irc.perl.org.

Tunnelling instructions

Use some GUI ssh software like PuTTY for Windows or SSHKeychain on a Mac. Or do it manually. 

ssh -l username_on_your_server -f -N -L 6667:irc.perl.org:6667 your_server

Then use "localhost" as your server.
